False hierarchy（偽の階層）

「False hierarchy（偽の階層）」は、ユーザーインターフェースやデザインにおいて、視覚的な要素やレイアウトを使って、本来の重要度や関連性が異なる情報や選択肢をあたかも同じように見せたり、逆に、重要度の低い要素を目立たせたりするダークパターンの一つです。

具体的な説明

False hierarchy（偽の階層）では、以下のような手法でユーザーを誤った認識に導くことが行われます。

  • 不正確な強調: 重要度が低い情報や選択肢を、フォントサイズや色、配置などによって不必要に目立たせ、重要だと感じさせる。
  • 選択肢の視覚的な均等化: 実際には重要度が異なる選択肢を、デザイン上同じように見せることで、どれを選んでも結果が同じように見える錯覚を与える。
  • 誤解を招くレイアウト: 重要な選択肢や情報が目立たない場所に配置され、ユーザーが誤った判断をしやすくなるようにする。
